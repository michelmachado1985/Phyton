# Phyton

Repositório com scripts básicos em Python para manipulação e análise de dados, utilizando principalmente **Pandas**, **Jupyter Notebook** e bibliotecas de machine learning.

---

## 📂 Conteúdo do Repositório

### 📓 Jupyter Notebooks (.ipynb)

| Arquivo | Descrição |
|---------|-----------|
| `Abrindo Arquivos de varias Fontes TXT, CSV, XLSX.ipynb` | Demonstra como abrir e carregar dados a partir de diferentes formatos de arquivo (TXT, CSV, XLSX). |
| `Alterando valores Nulos.ipynb` | Técnicas para identificar e tratar valores nulos em DataFrames. |
| `Excluindo NAN.ipynb` | Métodos para remover registros com valores NAN (Not a Number). |
| `Filtrando o dataframe.ipynb` | Como aplicar filtros para selecionar dados específicos em um DataFrame. |
| `Removendo linhas duplicadas.ipynb` | Identificação e remoção de linhas duplicadas em conjuntos de dados. |
| `Classificação_Árvores_de_Decisão.ipynb` | Introdução à classificação usando o algoritmo de Árvore de Decisão (Decision Tree). |

---

### 📄 Arquivos de Dados

| Arquivo | Descrição |
|---------|-----------|
| `Alunos.txt` | Arquivo de texto com dados de exemplo sobre alunos (usado nos scripts de manipulação). |
